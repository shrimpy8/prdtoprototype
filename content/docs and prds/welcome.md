# Welcome to Your Productivity Hub

This is your personal workspace for managing documents, PRDs, code, and prototypes.

## Getting Started

- **Docs**: Store your markdown documents here
- **PRDs**: Keep your product requirements documents
- **Code**: Organize code snippets and examples
- **Prototypes**: Quick interactive prototypes

## Features

- Browse files in the sidebar
- View markdown files with live rendering
- Run prototypes in a sandboxed environment
- Keep everything organized and local
